De ce site :

Ce site est un site démo, un projet amateur dans le but de prouver mes connaissances, en Angular.js, mais aussi en Twitter Bootstrap 3, en SCSS ...

J'ai démarré mon projet en suivant le Workflow du Dev front-end moderne : J'ai utilisé un generator Yeoman, celui d'Angular.js.

Grâce à Yeoman, on démarre un projet selon un standard partagé par la communauté. Diverses outils sont intégrés à la demande : Twitter Bootstrap 3, la suite de test Karma, le lanceur de tâche Grunt, le gestionnaire de paquet pour le développement web front end Bower. Tout est paramétré pour fonctionner "out of the box", et c'est un gain de temps exceptionnel, auquel j'adhère à 100%.
